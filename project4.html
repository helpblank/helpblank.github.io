Less Is More: Book Price Comparison

A website designed to help students find the cheapest sources for books.

Background

The user creates a list of books by inputting the ISBN or title of the book, which will be normalized through a series of regular expressions. Additionally, the user may input max or minimum prices for each book, to allow for further result refining. The list of books as well as price requirements are then passed to a JavaScript function that makes calls to multiple online store APIs (Google Books, DirectTextbooks, etc.) to find matches. The results are then shown as a list of links to the corresponding cheapest listings.
