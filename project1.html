Global Route and Task Generalization

This project was assigned as part of my senior capstone design course at Texas A&M, and the aim of the project was to design and implement a system for generation and user modification of behavioral state change triggers.

Background

In our project, we were provided with a set of HDMap data files, and rosbag containers, which contained data that would simulate the location of a car driving in realtime. Using that provided material, we needed to program a graphical interface that allows user to create, modify, copy triggers. We added functionality for the user to save and load node profiles, and once all aspects of the node system were completed, we needed to track the vehicle location and notify system of trigger activation.
